NihonGo Study

일본어 단어, 문장, 복습, 퀴즈 로직을 공통 패키지로 분리하는 중입니다. 이 페이지는 `apps/web` 정적 배포 경로가 정상적으로 연결되었는지 확인하는 웹 entry입니다.

core 학습 로직

검색, 필터링, 복습 큐, 퀴즈 문제 생성 같은 렌더링 전 공통 계산을 담당합니다.

ui 공통 화면

페이지 렌더러, 컴포넌트, 스타일, DOM patch 적용 유틸리티를 제공합니다.

storage 저장소 경계

desktop SQLite 구현은 분리됐고, 웹 IndexedDB 저장소는 다음 단계에서 추가할 수 있습니다.

tts 음성 재생

브라우저 TTS와 VOICEVOX 공통 로직을 앱별 shell에서 재사용할 수 있도록 분리했습니다.

desktop Electron 앱

현재 동작하는 데스크톱 앱은 `apps/desktop`에 유지하고 adapter만 얇게 남겼습니다.

web 웹 앱 준비

GitHub Pages는 이 `apps/web` 폴더를 그대로 배포합니다.

Static content is deployed from apps/web.
